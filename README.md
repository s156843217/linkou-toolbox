# 林口置產工具箱

一站式置產服務，整合以下工具於同一網站：

- 🏫 **學區快查** `/school/` — 查里鄰與國中小學區、地圖定位、幼兒園名額
- 🚌 **公車路線** `/bus/` — 查附近站牌與經過路線、完整走向
- 🏦 **房貸試算** `/mortgage/` — 薪資／月繳／頭期／自訂四種試算，附林口各地段行情
- 📄 **租約產生器** `/rent/` — 依內政部範本產生租賃契約，可列印 PDF／匯出 Word

純前端（HTML + CSS + 原生 JS），無建置流程，直接開 `index.html` 即可執行。
資料以新北市林口區為主，介面為繁體中文。

> 本站結果僅供參考，不構成財務或法律建議。
